# Expert Images

## Purpose
Expert photos for trust-building in questionnaire, main page, and consultation features.

## Usage
Import these images in Astro components using the public path:

```astro
<img src="/assets/experts/expert-1.png" alt="Expert" />
```

Or in React components:
```tsx
import expertImage from '/assets/experts/expert-1.png';
```

## Available Images

### 4:3 Aspect Ratio (Portrait)
- `expert-1.png` - Expert photo 1 (4:3 format)
- `expert-2.png` - Expert photo 2 (4:3 format)
- `expert-3.png` - Expert photo 3 (4:3 format)

### Landscape Format
- `expert-landscape-1.png` - Expert photo in landscape format
- `expert-landscape-2.png` - Expert photo in landscape format
